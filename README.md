# Cognión!

> Plataforma de cuestionarios para facultades de ingeniería

Cognión es una aplicación web de evaluación de conocimiento inspirada en la dinámica de Kahoot!, diseñada para el contexto universitario de carreras de ingeniería. Los docentes crean y publican cuestionarios; los alumnos los responden con corrección automática y resultados inmediatos.
